FEATURE — SERVICE ZONES BY ZIP CODE

Build a service zone system that color codes 
geographic coverage areas by zip code.
Zones appear throughout the platform wherever
jobs and clients are displayed.

Mobile principle: on mobile you are OPERATING 
not ADMINISTERING. Show colors only. Never show 
zip codes on mobile. Keep zone information to 
the absolute minimum needed to run the day.

---

PART A — DATABASE

Create table: service_zones
  id, company_id,
  name (text — e.g. "Southwest Zone"),
  color (text — hex color e.g. "#FF69B4"),
  zip_codes (text array),
  is_active (boolean, default true),
  sort_order (int),
  created_at (timestamptz)

Create table: service_zone_employees
  id, zone_id, user_id, company_id

Add to clients table:
  zone_id (uuid, nullable → service_zones)

Add to jobs table:
  zone_id (uuid, nullable → service_zones)

RLS on all tables via company_id.

Auto-assign zone: when a job or client is 
created or updated with an address, extract 
the zip code and match against 
service_zones.zip_codes array.
If match found: auto-set zone_id.
If no match: zone_id remains null.

PART F — ZIP CODE SCREENING ON QUOTE TOOL

On online quote iFrame and office quote form,
when zip code is entered:

If zip matches a zone:
- Show: "● Southwest Zone — We service 
  your area" in green
- Continue with quote

If zip does not match any zone:
- Show: "We don't currently service this area.
  Enter your email to be notified when 
  we expand."
- Email capture field + "Notify Me" button
- Save to waitlist table:
  company_id, email, zip_code, created_at
- Do not proceed with quote

If no zones configured:
- Skip zip screening entirely

MOBILE quote form:
- If in zone: show small green dot + 
  "We service this area" — no zone name
- If out of zone: same message as desktop

PART I — GLOBAL MOBILE RULE FOR ZONES

Apply this rule everywhere zones appear on mobile:

SHOW: Colored dot or colored border only
NEVER SHOW on mobile:
- Zone name text
- Zip code lists
- Zone performance data
- Cross-zone warnings
- Zone assignment details

The color communicates everything needed 
while operating in the field.
Full zone management is desktop only.
