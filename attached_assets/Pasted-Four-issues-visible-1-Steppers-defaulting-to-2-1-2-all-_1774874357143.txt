**Fix 4 — Add-on checkboxes all functional:**

Only Refrigerator is currently selectable. Fix all 5 add-on checkboxes:

Each add-on card must have its own independent checked state:

```javascript
const [selectedAddons, setSelectedAddons] = useState({
  oven: false,
  refrigerator: false,
  kitchenCabinets: false,
  windows: false,
  cleanBasement: false
});

const toggleAddon = (key) => {
  setSelectedAddons(prev => ({
    ...prev,
    [key]: !prev[key]
  }));
};
```

Each card's checkbox and the entire card surface must be clickable — tapping anywhere on the card toggles it. Do not rely on the checkbox input click event alone — wrap the entire card in an onClick handler.

```jsx
<div 
  onClick={() => toggleAddon('oven')}
  style={{cursor: 'pointer'}}
>
  {/* card content */}
  <input 
    type="checkbox" 
    checked={selectedAddons.oven}
    onChange={() => toggleAddon('oven')}
    onClick={(e) => e.stopPropagation()}
  />
</div>
```

Apply this pattern to all 5 cards. Confirm each card independently toggles without affecting others.
