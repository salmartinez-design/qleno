PROMPT — NOTIFICATION CENTER WITH 3 ALERT TYPES
Add a notification bell to the Qleno top nav that opens a notification dropdown. Three notification types only — new bookings, late clock-ins, and unassigned jobs within 48 hours.

STEP 1 — DATABASE
sqlCREATE TABLE IF NOT EXISTS notifications (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  company_id integer NOT NULL,
  type varchar(50) NOT NULL,
  title varchar(255) NOT NULL,
  body text,
  link varchar(500),
  read boolean DEFAULT false,
  created_at timestamptz DEFAULT now()
);

CREATE INDEX IF NOT EXISTS idx_notifications_company_unread 
  ON notifications(company_id, read, created_at DESC);

STEP 5 — AUTO-GENERATE NOTIFICATIONS
Trigger 1 — New booking (new_booking):
When a booking is submitted via the widget and a job record is successfully created, immediately insert a notification:

Title: New online booking — [first_name] [last_name]
Body: [service_type] · [scheduled_date formatted] · [arrival_window] · $[first_visit_total]
Link: /clients/[client_id]
company_id: from branchConfig (Oak Lawn or Schaumburg based on zip)

Trigger 2 — Late clock-in (late_clockin):
When a tech clocks into a job more than 20 minutes after the scheduled job start time, insert a notification:

Title: Late clock-in: [tech first name] — [client last name]
Body: Scheduled [scheduled_time], clocked in at [actual_clockin_time] ([X] min late)
Link: /jobs/[job_id]
company_id: from job record

Trigger 3 — Unassigned job within 48 hours (job_unassigned):
Add to the existing nightly 2AM cron. Query jobs where:

status = 'scheduled'
scheduled_date is within the next 48 hours
No technician assigned in job_technicians for this job
No existing unread job_unassigned notification already exists for this job_id

Insert a notification:

Title: Unassigned job on [formatted date] — [client last name]
Body: [service_type] · [arrival_window] · $[first_visit_total] — needs a technician assigned
Link: /jobs/[job_id]
company_id: from job record
