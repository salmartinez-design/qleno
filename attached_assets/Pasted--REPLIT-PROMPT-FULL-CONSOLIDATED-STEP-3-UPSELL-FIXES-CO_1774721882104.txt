**FIX 2 — UPSELL CARD PRICE CALCULATION**

The price is showing $0.00. Fix:

```javascript
// Calculate recurring rate using the same pricing engine as the quote builder
// Use name-based scope lookup — never ID-based
const baseRecurringPrice = await calculateQuotePrice({
  scopeName: 'recurring cleaning',
  sqft: enteredSquareFootage,
  bedrooms: enteredBedrooms,
  bathrooms: enteredBathrooms
});

// Cadence multipliers — read from pricing engine
const cadenceMultipliers = {
  'weekly': 0.80,
  'biweekly': 0.90,
  'every_4_weeks': 1.00
};

const recurringRate = baseRecurringPrice * cadenceMultipliers[selectedUpsellCadence];

// Pull discount % from offer_settings
const firstVisitRate = recurringRate * (1 - (offerSettings.upsell_discount_percent / 100));
```

Price updates in real time as cadence pill is selected. Uses `offer_settings.upsell_discount_percent` and `offer_settings.rate_lock_duration_months` — never hardcoded values.
