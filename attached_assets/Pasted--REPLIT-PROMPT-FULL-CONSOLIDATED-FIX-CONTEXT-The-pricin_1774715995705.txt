**FIX 1 — UPSELL CARD PRICE CALCULATION**

The upsell card is showing $0.00 for the recurring rate. Fix the price calculation without touching the pricing engine tables.

When the customer selects a cadence pill in the upsell card (Weekly / Every 2 Weeks / Every 4 Weeks), calculate the recurring rate as follows:

```javascript
// Use the same pricing calculation function used in the quote builder
// Pass scope: 'recurring cleaning' (name-based, not ID-based)
// Pass sqft: the value entered in the Square Footage field on this step
// Pass cadence: the selected cadence from the upsell card

const basePrice = await calculateQuotePrice({
  scopeName: 'recurring cleaning',
  sqft: enteredSquareFootage,
  bedrooms: enteredBedrooms,
  bathrooms: enteredBathrooms
});

// Apply cadence multipliers (read from pricing engine, do not hardcode)
// Weekly = 0.80x, Every 2 Weeks = 0.90x, Every 4 Weeks = 1.00x
const cadenceMultipliers = {
  'weekly': 0.80,
  'biweekly': 0.90,
  'every_4_weeks': 1.00
};

const recurringRate = basePrice * cadenceMultipliers[selectedCadence];

// Pull discount from offer_settings (default 15%)
const firstVisitRate = recurringRate * (1 - (offerSettings.upsell_discount_percent / 100));
```

Display in the upsell card:
- Primary line (`#1A1917`, 15px, semi-bold): "Your first recurring cleaning: ~~$[recurringRate formatted]~~ **$[firstVisitRate formatted]**"
- Secondary line (`#6B6860`, 12px): "Then $[recurringRate formatted]/visit — locked for [rate_lock_duration_months] months."

Both values update in real time as cadence changes. If sqft is 0 or not entered, show "Enter your home size above to see your rate" in secondary text instead of a price.
